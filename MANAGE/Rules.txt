++=================|-|-|-|-|__**RULES**__|-|-|-|-|=================++

__**PASAL 1 (ATURAN UMUM)**__
```
1. DATANG TEPAT WAKTU UNTUK JADWAL WAJIB. PEMBAHASAN AKAN DIMULAI PADA JAM YANG SUDAH DITENTUKAN. KETINGGALAN TIDAK ADA PENGULANGAN.
2. WAKTU KELOMPOK BERDASARKAN YANG TELAH DISEPAKATI.
3. HAK PEMBATALAN KELOMPOK DIHARI ITU HANYA DIMILIKI OLEH FOUNDER.
4. TIDAK ADA PEMBAHASAN LAIN SELAIN APA YANG SUDAH DISEPAKATI.
5. TIDAK AKAN ADA PEMBUBARAN KELOMPOK. BUKAN BERARTI TIDAK ADA PERGANTIAN ANGGOTA.
6. GUNAKAN FASILITAS UMUM UNTUK KEPENTINGAN UMUM.
```

__**PASAL 2 (ETIKA DAN SIKAP)**__
```
1. DI IZINKAN UNTUK FLEXING JIKA ITU MENJADI ALASAN SEMANGAT ANDA (TANPA MENUNJUKKAN IDENTITAS ANGGOTA LAIN).
2. JANGAN MERUBAH APAPUN PADA CHANEL DISCORD TANPA KEPENTINGAN.
3. GUNAKAN WAKTU SERIUS UNTUK SERIUS, GUNAKAN WAKTU BERCANDA UNTUK BERCANDA.
4. MATERI AKAN DIBERIKAN TANPA PEMBERITAHUAN TERLEBIH DAHULU.
5. TIDAK ADA PENGULANGAN PENJELASAN ATAS KETIDAK HADIRAN.
6. TIDAK ADA PENGULANGAN PENJELASAN KETIKA TERJADI KELALAIAN DALAM MENDENGARKAN.
7. DILARANG SPILL YANG ADA DIDALAM KELOMPOK SECARA VOCAL, VISUAL MAUPUN VERBAL.
8. TURUNKAN EGO (DALAM HAL APAPUN) UNTUK KEPENTINGAN BERSAMA.
9. DAHULUKAN YANG PENTING. JIKA MENURUTMU HAL INI TIDAK PENTING, LEBIH BAIK JANGAN LANJUT.
10. TEKAN EMOSI ANDA UNTUK DIRI ANDA SENDIRI. JANGAN SAMPAI SEKITAR ANDA TERPENGARUH OLEH EMOSI ANDA.
11. TIDAK ADA PEMBAHASANDAN ATAU MENGERJAKAN TUGAS KULIAH SAAT JADWAL KODING. DAN TIDAK ADA PEMBAHASAN DAN ATAU MENGERJAKAN KODING SAAT TUGAS KULIAH.
12. KALIAN BICARA SAYA DIAM. KALIAN DIAM SAYA MENJELASKAN.
13. SETIAP PERTANYAAN YANG BERHUBUNGAN DENGAN TASK KODING TIDAK AKAN DIJAWAB PADA JADWAL WAJIB.
```

__**PASAL 3 (TUGAS DAN KUALITAS ANGGOTA)**__
```
1. TIDAK ADA DISKUSI UNTUK KEHADIRAN. YANG BISA SILAHKAN, YANG TIDAK BISA RESIKO KETINGGALAN.
2. PENGERJAAN TASK TIDAK MENJADI SYARAT UNTUK KELANJUTAN MATERI. KETINGGALAN LEBIH DARI 2 TASK SILAHKAN MENGUNDURKAN DIRI.
3. NIAT DAN KESERIUSAN ANDA TIDAK BERPENGARUH TERHADAP KULIAH ANDA. TAPI KALAU TIDAK SERIUS MENDING TIDAK USAH SEKALIAN.
4. UPLOAD CODINGAN TASK KE GITHUB (RAPIKAN FOLDER SESUAI KETENTUAN) DAN SCREENSHOT HASIL KE DISCORD.
5. UNTUK MENGEJAR KEEFISIENAN MAKA DALAM 2 MINGGU HARUS MENYELESAIKAN 3 TASK.
6. TIDAK ADA JAWABAN "MALAS" UNTUK SETIAP TUGAS ATAU PERINTAH DARI FOUNDER SEHUBUNGAN DENGAN KELOMPOK INI.
```

*```
NOTE : JIKA SUDAH MASUK KEDALAM ATURAN. SANKSI DARI PELANGGARAN DITANGGUNG SENDIRI. SANKSI TERBERAT AKAN DILAKUKAN KICK OUT SECARA SEPIHAK. TIDAK DAPAT MENGIKUTI ATURAN, TIDAK MENGIKUTI KELOMPOK BISA MENJADI PILIHAN ANDA. TIDAK ADA YANG MEMAKSA UNTUK TERUS LANJUT. SEBELUM LANJUT MATERI JAVASCRIPT DAN YANG LAINNYA AKAN DILAKUKAN PENYARINGAN ANGGOTA (KUALIFIKASI BERDASARKNA NIAT. NIAT DAPAT DILIHAT DARI HASIL PEKERJAAN DAN KETEPATAN WAKTU). ATURAN BISA BERUBAH SESUAI KONDISI LAPANGAN.
```*
